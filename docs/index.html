Virtual Conductor

A Better Metronome for Orchestra Musicians

About Virtual Conductor

Virtual Conductor is a metronome geared towards orchesta, choir, and band musicians. In these musical setting, a conductor leads the ensemble using the motions of their hands. Virtual Conductor mimics this motion in order to track the beat in a way that is meaningful to these musicians while they practice.
